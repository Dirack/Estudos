## Variáveis

Utilizar variáveis

```sass
$cor-padrao: #EEE

.quadrado{
  background-color: $cor-padrao
}
```

## Grupos de elementos

Dá para agrupar propriedades dentro de elementos

```sass
nav {
  background-color: #EEE;

  ul {
    color: white;
  }

  li {
    display: inline-block;
  }
}
```
