# Aspectos gerais de configuração do TypeScript

## Compilação do arquivo em watchmode

```
tsc src/script.ts -w
```

## Criando arquivo de configuração do TypeScript

```
tsc --init
```

## Compilação do projeto utilizando o arquivo de configuração do TypeScript

```
tsc
```
